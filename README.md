Este repositório representa as minhas primeiras linhas de códigos de minha autoria, sem me basear em outros prompts. Além de ser, não só o primeiro projeto da faculdade como também o que me aproximou de meus amigos. O código basicamente consiste em um formulário de adoção canina, onde você deve preencher uma lista de requisitos para termos uma noção de qual seria o seu cão ideal. 
E para mostrarmos que estávamos empenhados em criar nossa própria "PetShop" até criamos um nome e um mascote para nosso estabelecimento, que carinhosamente chamamos de "AUmigos da vizinhança", que logo se tornou até o nome do nosso grupo.
